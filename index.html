Project-1: RainyDays

Rainydays

RainyDays is an interactive e-commerce website for men’s and women’s jackets, built using JavaScript, HTML, and CSS. It dynamically fetches product data from an API, enabling users to browse the catalog, filter by gender, and manage items in a shopping cart with a smooth, user-friendly interface.

Please check out the website and GitHub

Project-2: Community Science Museum

Community Science Museum

The Community Science Museum website is a fun and educational platform designed for children and families. It highlights interactive exhibits, events, and visitor info in a colorful, easy-to-use layout. The site is fully responsive and accessible, encouraging curiosity and visits to the museum.

Please check out the website and GitHub

Project-3: Cakes & Comfort

Cakes & Comfort

Cakes & Comfort is a delightful and interactive blog website where users can explore “Sprinkles’” favorite sweet treat recipes. Visitors can browse a variety of mouth-watering desserts, and by registering, they can create and share their own favorite recipes with the community.

Please check out the website and GitHub
